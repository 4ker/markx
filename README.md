# Markx

*Marx* is a [Markdown] editor with [Pandoc] flavor in your browser.

## Features

1. Zenware
2. Display math - $x^2+y^2=r^2$
3. Code highlight
4. Markdown flavor of Pandoc

## Technology

  * [Python] with [Flask]
  * [PageDown] markdown editor
  * [Twitter Bootstrap] with [Glyphicons Free]
  * [Google Code Prettifier]
  * [MathJax]
